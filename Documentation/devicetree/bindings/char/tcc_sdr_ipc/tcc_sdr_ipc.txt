Telechips IPC Driver for SDR Applications

Required properties:
	-compatible:
    	telechips,sdr_ipc

	-device-name:
		device driver name

	-mbox-names:
		List of identifier strings for each mailbox channel.

	-mboxes:
		List of phandle and mailbox channel specifiers.

	-mbox-id:
		TCC_MBOX_SDR_ID

Example:
	sdr_ipc {
		compatible = "telechips,sdr_ipc";
		device-name = "tcc_sdr_ipc";
		mbox-names ="sdr-ipc-a72";
		mboxes = <&cb_mbox TCC_CB_MBOX_SDR>;
		mbox-id = TCC_MBOX_SDR_ID;
	};
