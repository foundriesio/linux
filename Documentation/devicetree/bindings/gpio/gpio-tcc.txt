Required properties:
 - compatible:
        "telechips,tcc-gpio"
 - reg: The gpio address range, relative to the gpio range
	<cell num>
        1 : gpio base address(ap)
        2 : range of gpio base address
 - gpio-controller : Indicates this device is a GPIO controller
 - #gpio-cells :
    <cell num>
         1 : pin number
         2 : polarity
              <value>
                [0] : for active high
                [1] : for active low
 - reg-offset: Offset from gpio base address
 - source-num: External interrupt source number of gpio pin
               Multiple ranges can be registered
    <cell num>
       1 : Count of source number ranges
//First source number ranges
       2 : start pin number
       3 : base source number
       4 : number of pin
//Second number ranges
       5 : start pin number
       6 : base source number
       7 : number of pin
//Next number ranges
      ...
 - pin-info:
    <cell num>
       1 : offset from gpio base address
       2 : start pin number
       3 : count of gpio pin
 - interrupt-cells: Should be two. The first cell is the GPIO Number. The
                    second cell is used to specify flags. The following
                    subset of flags is supported:
     - trigger type:
       1 = low to high edge triggered
       2 = high to low edge triggered
       4 = active high level-sensitive
       8 = active low level-sensitive
 - interrupt-controller: Identifies the node as an interrupt controller
 - interrupts: The interrupt number to fthe cpu.The interrupt specifier format
               depends on the interrupt controller

Optional properties:
