 * Bindings for Telechisp UFS driver

 Required properties:

 - compatible: must be "telechips,tcc806x-sc-ufs"
 - sc-firmware: protocol driver to communicate with storage core

 Example:

 tcc_sc_ufs: tcc_sc_ufs {
     compatible = "telechips,tcc805x-sc-ufs";
     status = "okay";
     sc-firmware = <&tcc_sc_fw>;
 };
