Pin controller node:
Required properties:
 - compatible:
		"telechips,tcc803x-pinctrl"
 - reg: The pinctrl(gpio) address range, relative to the pinctrl range
	<cell num>
        1 : gpio base address(ap)
        2 : range of gpio base address
        3 : pmgpio base address(micom)
        4 : range of pmgpio base address
 - interrupts: The interrupt number to the cpu.The interrupt specifier format 
               depends on the interrupt controller

GPIO controller/bank node:
Required properties:
 - gpio-controller : Indicates this device is a GPIO controller
 - #gpio-cells :
    <cell num>
         1 : pin number
         2 : polarity
              <value>
                [0] : for active high
                [1] : for active low
 - reg: The gpio address range, relative to the pinctrl range
    <cell num>
        1 : offset from pinctrl base address
        2 : start pin number
        3 : pin number
 - source-num: External interrupt source number of gpio pin
               Multiple ranges can be registered
    <cell num>
       1 : Count of source number ranges
//First source number ranges
       2 : start pin number
       3 : base source number
       4 : number of pin
//Second number ranges
       5 : start pin number 
       6 : base source number
       7 : number of pin
//Next number ranges
      ...


Optional properties:
 - telechips,no-pull : Disable pull-up/down
 - telechips,pull-up : Select pull-up and enable pull-up
 - telechips,pull-down : Select pull-down and enable pull-down
 - telechips,input-enable : Disable ouput
 - telechips,output-low : Enable output and set to low
 - telechips,output-high : Enable output and set to high
 - telechips,input_buffer_enable : Enable input buffer
 - telechips,input_buffer_disable : Disable input buffer
 - telechips,schmitt-input : Set to Schmitt input type
 - telechips,cmos-input : Set to CMOS input type
 - telechips,slow-slew : Select slew rate to fast
 - telechips,fast-slew : Select slew rate to low
 - telechips,eclk-sel : Select external input clock
 - telechips,drive-strength : Set port driver strength
