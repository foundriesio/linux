* Temperature Sensor on Telechips SoCs

- compatible : "telechips,tcc-thermal";
- reg : physical base address of the controller and length of memory mapped
	region.
- interrupts : The interrupt number to the cpu.The interrupt specifier format
	depends on the interrupt controller
- throttle_count : Number of throttling table.
- throttle_passive_count : The number of throttle_tables associated with
	throttling.
	Thermal sensor driver supports throttling function.
- cal_type : Type of calibration.
	1. TCC802x/TCC803x/TCC897x support one-point trimming calibration
		TYPE_ONE_POINT_TRIMMING
	2. TCC805x supports two-points trimming calibration
		TYPE_TWO_POINT_TRIMMING
- throttle_table : Operation table according to the temperature
	temp means temperature for operation.
	freq_max_cluster0 is the maximum CPU clock depending on the temperature.
	If it reaches the temperature, the throttling function lowers the CPU
	clock.
- polling-delay-idle : The maximum number of milliseconds to wait between polls
	when checking this thermal zone.
	If you want to use polling check (for throttling or shutdown function),
	set this value.
- polling-delay-passive : The maximum number of milliseconds to wait between
	polls when performing passive cooling (throttling).

Example:
Property for TCC805x

- core : A property to indicate whether or not to initialize the thermal driver.
	If the core value is “CPU0”, it is recognized as the primary core and
	the thermal configuration register is set.
- interval_time : Period at which the thermal sensor captures the temperature in
	continuous mode.

Example(TCC802x/TCC803x/TCC897x/TCC899x/TCC901x):

&thermal {
        compatible = "telechips,tcc-thermal";
        reg = <0x14700000 0x4 0x14700030 0x4 0x14200290 0x4 0x1420029C 0x4
               0x14200298 0x4 0x1470000C 0x4 0x14700010 0x4>;

        status = "okay";

        throttle_count = <5>;
        throttle_active_count = <0>;
        throttle_passive_count = <5>;
        interrupts = <GIC_SPI 40 0>;
        cal_type = "TYPE_ONE_POINT_TRIMMING";
        polling-delay-idle = <0>;
        polling-delay-passive = <0>;

        threshold_temp = <85>;

        throttle_table {
                throttle_tab_0 {
                        temp = <75>;
                        freq_max_cluster0 = <1000000>;
                        freq_max_cluster1 = <900000>;
                };
                throttle_tab_1 {
                        temp = <80>;
                        freq_max_cluster0 = <900000>;
                        freq_max_cluster1 = <810000>;
                };
                throttle_tab_2 {
                        temp = <85>;
                        freq_max_cluster0 = <800000>;
                        freq_max_cluster1 = <720000>;
                };
                throttle_tab_3 {
                        temp = <90>;
                        freq_max_cluster0 = <700000>;
                        freq_max_cluster1 = <630000>;
                };
                throttle_tab_4 {
                        temp = <95>;
                        freq_max_cluster0 = <600000>;
                        freq_max_cluster1 = <540000>;
                };
        };
};

Example(TCC805x):
thermal: thermal@14700000 {
	compatible = "telechips,tcc-thermal-805x";

	sc-firmware = <&tcc_sc_fw>;

	reg = <0x0 0x14700000 0x0 0x160>,//0:Enable register

	status = "okay";

	throttle_count = <4>;
	throttle_active_count = <0>;
	throttle_passive_count = <4>;
	interrupts = <GIC_SPI 40 IRQ_TYPE_LEVEL_HIGH>;
	cal_type = "TYPE_TWO_POINT_TRIMMING";
	polling-delay-idle = <1000>;
	polling-delay-passive = <1000>;
	core = "CPU0";

	threshold_high_temp = <75>;
	threshold_low_temp = <(-40)>;
	interval_time = <262143>;

	throttle_table {
		throttle_tab_0 {
			temp = <75>;
		};
		throttle_tab_1 {
			temp = <85>;
		};
		throttle_tab_2 {
			temp = <95>;
		};
		throttle_tab_3 {
			temp = <105>;
		};
	};
};
