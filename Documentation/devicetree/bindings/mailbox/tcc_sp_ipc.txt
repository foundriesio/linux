This is for "Telechips sp-ipc device"

Required properties:
	- compatible:
		- CORE <-> M4(SP): telechips,sp
	-reg:
		Specifies base physical address and size of the registers map.

Optional properties:
	- mailbox-tcc:
		Telechips Mailbox device Telechips Mailbox device
		This device file must be enable to use sp_ipc.

Example:

	sec-ipc-m4 {
		compatible = "telechips,sec-ipc-m4";
		mbox-names = "sec-ipc-m4";
		mboxes = <&sec_ipc 0>;
		mbox-id = TCC_MBOX_HSM_ID;
	};
